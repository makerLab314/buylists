🔐 GitHub Authentifizierung

Um Listen zu erstellen oder zu bearbeiten, benötigst du ein GitHub Personal Access Token mit "repo" Berechtigung.

So erstellst du ein Token:
1. Gehe zu GitHub Token Settings
2. Wähle "Generate new token (classic)"
3. Gib einen Namen ein (z.B. "Einkaufslisten Admin")
4. Wähle den Scope "repo" (voller Zugriff auf Repositories)
5. Klicke auf "Generate token" und kopiere den Token
